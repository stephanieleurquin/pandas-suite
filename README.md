# Projet Pandas Suite

Ce projet contient des scripts Python pour pratiquer le nettoyage et l'analyse de données avec Pandas.

## Scripts principaux

- `pandasuite.py` : 
  - Lit un fichier CSV
  - Extrait les noms et âges avec des expressions régulières
  - Remplace les valeurs manquantes par "inconnu"
  - Affiche le résultat dans un DataFrame propre

## Objectifs du projet

- Apprendre à manipuler des DataFrames avec Pandas
- Nettoyer et transformer des données
- Utiliser Git et GitHub pour versionner le code

## Technologies utilisées

- Python 3.x
- Pandas
- Expressions régulières (Regex)
